Issues Found & Fixed

📦

1. Chunk Size

2048 → 512

128ms → 32ms buffer. First-word cut by 96ms. 4× more frames per second, each only 512 Int16 samples.

pcm-capture-worklet.js
⏯️

2. Playback Drift

Cumulative → Adaptive

Cursor reset at currentTime + 50ms if drift exceeds 300ms. Eliminates cumulative scheduling backlog.

voiceLive.js
📊

3. Noise Cal

30 → 10 frames

~500ms → ~167ms startup calibration. Faster orb response without accuracy loss.

VoiceView.vue
🔄

4. Level Meter Teardown

Double-close → Safe

stopLevelMeter() always owned its own AudioContext. No risk of closing a shared context mid-flight.

VoiceView.vue

What was removed and why:
  • AudioContext reuse — caused audio artifacts ("3 vocal"). Level meter kept its own context.
  • Client-side VAD gate — clipped first speech frames, produced robotic cutoff. Rely on Gemini's server VAD.

📁 Files Changed

  • cmd/desktop/frontend/public/pcm-capture-worklet.js — chunkSize 2048→512
  • cmd/desktop/frontend/src/lib/voiceLive.js — adaptive playback cursor
  • cmd/desktop/frontend/src/components/VoiceView.vue — NOISE_CAL_FRAMES 30→10, clean teardown
